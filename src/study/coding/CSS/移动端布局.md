# 2 vw适配方案
## 2.1 vw和vh基本使用
- `vw`和`vh`是相对单位，相对视口尺寸计算结果
- `vw`：`viewport width`（1vw = 1/100视口宽度 ）
- `vh`：`viewport height` ( 1vh = 1/100视口高度 )
## 2.2 vw布局
- `vw`单位的尺寸 = `px` 单位数值 / ( 1/100 视口宽度 )
## 2.3 vh问题
- `vh`是1/100视口高度，全面屏视口高度尺寸大，如果混用可能会导致盒子变形
